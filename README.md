# RK3588-STM32-ROS2-SLAM-Robot
基于 RK3588 与 STM32 异构架构的 ROS2 自主导航机器人

## 硬件

| 层级            | 核心硬件                                                  | 核心功能                             |
| --------------- | --------------------------------------------------------- | ------------------------------------ |
| 主控芯片        | RK3588 开发板（含 NPU）                                   | ROS2 建图导航、AI 视觉、传感器融合   |
| 下位机 MCU      | STM32F407VET6                                             | 电机闭环控制、IMU 姿态解算、CAN 通信 |
| 机械运动        | 四驱差速编码器电机小车 + TB6612 驱动                      | 自主移动、里程计数据采集             |
| SLAM & 视觉感知 | 1. 思岚 RPLIDAR A1 激光雷达2. 奥比中光 Astra Pro 深度相机 | 360° 环境扫描、近距离避障、视觉感知  |
| IMU 姿态传感器  | MPU9250 9 轴 IMU                                          | 姿态检测、航向定位、EKF 融合数据源   |
| 通信层          | TJA1050 CAN 总线模块                                      | 上下位机高速稳定通信                 |
| 电源            | 12V 10A 21000mAh 锂电池                                   | 整机供电                             |
